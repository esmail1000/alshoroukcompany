# Alshorouk Company Website

موقع احترافي لشركة الشروق لمواد البناء باستخدام Next.js + TypeScript + Tailwind CSS.

## المرحلة الحالية

هذه النسخة تشمل تطوير المرحلة الثانية:

- تحسين الهيدر وإضافة شريط بيانات سريع في الديسكتوب.
- تحسين Hero Section وربط زر التواصل بالواتساب.
- إضافة نقاط ثقة داخل البانر الرئيسي بدون أيقونات.
- تحسين كروت المنتجات.
- تحسين كروت المميزات بدون أيقونات، مع ترقيم بسيط فقط.
- إضافة Metadata لكل لغة لتحسين SEO.
- ضبط بعض تفاصيل التصميم والـ responsive.

## التشغيل

```bash
npm install
npm run dev
```

افتح المتصفح على:

```bash
http://localhost:3000/ar
```

وللإنجليزي:

```bash
http://localhost:3000/en
```

## الصفحات

- `/ar` و `/en` الصفحة الرئيسية
- `/ar/about` و `/en/about` من نحن
- `/ar/products` و `/en/products` المنتجات
- `/ar/gallery` و `/en/gallery` المعرض
- `/ar/contact` و `/en/contact` التواصل
- `/ar/articles` و `/en/articles` المقالات

## تعديل بيانات التواصل

افتح الملف:

```bash
lib/site.ts
```

وعدّل أرقام الهاتف، واتساب، البريد الإلكتروني، العنوان، وروابط السوشيال.

## تعديل المحتوى والصور

افتح الملف:

```bash
data/content.ts
```

وعدّل النصوص، المنتجات، الصور، والمقالات.
